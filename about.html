About Khasi-English Translator

Smart. Simple. Built for real users to bridge the gap between Khasi and English communication.

What is this app?

The Khasi-English Translator App is a modern mobile application crafted for users who want a quick, accurate, and user-friendly way to translate between Khasi and English. Whether you're a student, tourist, or native speaker, this app provides seamless translations with a sleek interface.

Key Features

  • Live translation between Khasi and English
  • Save your favorite translations
  • Track your history with timestamps
  • Easy sharing of translations via apps
  • Secure login with Google or Email
  • Text-to-speech for accurate pronunciation
  • Works offline (with SQLite for guests)

Technology Behind the App

The app uses Google Cloud Translation API for accurate results, Firebase for secure user login and real-time database, and SQLite for storing offline data for guest users. It is built entirely in Java for Android and supports both light and dark UI modes.

Who Can Use This?

This app is ideal for:

  • Khasi speakers learning English
  • English speakers wanting to learn Khasi
  • Students, travelers, researchers, and locals
